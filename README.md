# blog

## 主要功能
- 响应式布局，适合手机端和PC端
- 特殊的欢迎页面
- 文章展示
- 标签搜索
- 

## 依赖
- papercss css主体框架
- animate.css css动画框架
- css.gg css图标库
- vuex 数据仓库，全局共享数据管理
- vue-router 前端路由管理
- axios 网络请求

## 困惑
- 响应式布局多种实现方式
